Projeto de conclusão da matéria de Programação para Web, 
realizada pelos alunos do segundo ano de informatica do Instituto Federal de Educação, Ciência e Tecnologia de São Paulo - Campus Jacareí

Este projeto tem como objetivo conectar pessoas e promover um ambiente seguro para a comunicação.
Ele foi desenvolvido para gerar interações sociais, falar do cotidiano e possibilitar a liberdade de expressão.
Utilizando as seguintes tecnologias: javaScript, Node.js, Express.js, bcryptjs, sequelize, MVC (Model-View-Controller), Async/Await, Sessions, Middleware.

Siga os passos abaixo para instalar e executar o projeto localmente:
- Clone o repositorio:
https://github.com/yleizarg07/projeto2semestre.git

- As dependências já vem instaladas, caso algo falte são elas:
npm install express bcryptjs sequelize ejs express-session

- Execute o projeto:
node server.js
